# 랜덤 발표자 추출 웹페이지

## 개요

수업 시간에 발표자를 랜덤으로 뽑아서 보여주는 웹페이지

---

## 목표 및 대상

중학교 학생

---

## 핵심 기능

- 발표자를 랜덤으로 뽑아서 보여주는 기능
  - 뽑을 인원 선택 기능
- 학생 명단 추가 기능
  - 삭제 기능
  - 한 명씩 추가 하는 기능과 여러 명을 일괄로 추가하는 기능
- 교사가 임의로 드러나지 않게 발표자를 정하는 기능

---

## 화면 구성

발표자를 보여줄 때 룰렛처럼 보여주는 화면

---

## 기술 스택

- 프론트엔드
  - react + vite

---

## 범위 (할 것 vs 안 할 것)

### Out-of-Scope
- DB, 백엔드 제외
